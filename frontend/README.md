# NextBloom Frontend

Next.js frontend for NextBloom e-commerce platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file:
```bash
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

3. Run development server:
```bash
npm run dev
```

Frontend will be available at http://localhost:3000

## Build

```bash
npm run build
npm start
```

## Features

- Product listing and search
- Product details
- Shopping cart
- Checkout process
- User authentication
- Order history
- Responsive design

## Tech Stack

- Next.js 14
- React 18
- Redux Toolkit
- Tailwind CSS
- TypeScript
- Axios

## Project Structure

- `app/` - Next.js app directory (pages)
- `components/` - React components
- `store/` - Redux store and slices
- `lib/` - Utility functions
- `public/` - Static assets
